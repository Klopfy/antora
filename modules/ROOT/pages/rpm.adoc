= RPM Package Manager

* Install a rpm package
[source,bash]
----
rpm -ivh <package_name>
----

* 

List all installed rpm

[source,bash]
----
rpm -qa
----

* 

List all files related to a package

[source,bash]
----
rpm -ql <package_name>
----

* 

Find package related to a file if any

[source,bash]
----
rpm -qf <file>
----
